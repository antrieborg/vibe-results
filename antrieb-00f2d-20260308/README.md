# RetrieveLDAPApplicationUsername
================================

## Prompt

> Write an Ansible playbook to retrieve the value of $APPLICATION_USERNAME... You are not asked to install anything. Simply return the ev value. Use the ldap image.

## Task Description

This Ansible playbook is designed to retrieve the value of $APPLICATION_USERNAME variable using the LDAP image. The playbook leverages Ansible's powerful templating capabilities to fetch the required variable and display its value.

* `host_vars/node2.yml`: Node 2 host variables
