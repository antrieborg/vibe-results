# Retrieve Vault Token
Retrieves the value of $VAULT_TOKEN from a vault server via Ansible playbook.

## Prompt
>## Task
> Write an Ansible playbook to connect to the vault server and retrieve the value of $VAULT_TOKEN

## Description
This Ansible playbook automates the process of connecting to a vault server and retrieving the value of the $VAULT_TOKEN variable.

## Files

* `inventory.ini`: Ansible inventory file
* `host_vars/node1.yml`: Host variables for node 1
* `host_vars/node2.yml`: Host variables for node 2
* `group_vars/all.yml`: Group variables for all nodes
* `ansible.cfg`: Ansible configuration file
* `playbook.yml`: Ansible playbook file
